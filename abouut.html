Banana Disease Detection Project

Empowering farmers with cutting-edge technology for disease detection in banana plants.

About Our Project

Our Banana Disease Detection project aims to assist farmers and researchers in identifying and addressing diseases in banana plants efficiently. This innovative solution leverages cutting-edge machine learning techniques, a robust backend, and a user-friendly frontend to provide accurate and practical disease analysis.

1. Machine Learning (ML)

The core of our project is a sophisticated ML model designed to classify banana plant diseases with high accuracy. We utilized:

  • Convolutional Neural Networks (CNN): To extract features and classify diseases from leaf images.
  • Region-Based Active Contour Segmentation: To preprocess images and highlight disease-affected regions.
  • Advanced Metrics and Techniques: Including normalization, resizing, and label encoding for optimal model performance.

Our model achieved an impressive 95% accuracy, distinguishing diseases such as Bacterial Soft Rot, Black Sigatoka, and Yellow Sigatoka, among others.

2. Backend

The backend of our application integrates the ML model with a Flask framework, enabling efficient image processing and seamless communication with the user interface. It handles:

  • Image uploads and preprocessing.
  • Disease prediction using the trained model.
  • JSON-based responses for user queries.

3. Frontend

Our simple and intuitive web interface allows users to:

  • Upload images of banana leaves.
  • Receive instant predictions of potential diseases.
  • Access a clear and accessible platform designed to maximize usability.

With this project, we aim to bridge the gap between technology and agriculture, offering a practical tool to combat plant diseases and enhance productivity.
